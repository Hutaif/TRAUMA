Classification
  • Schatzker: descriptive
    Type I: transverse, occurs at sigmoid notch
    Type II: transverse-impacted
    Type III: oblique, hyper-extension injury extending from sigmoid notch and running distally
    Type IV: oblique-distal, extends distal to coronoid leading to instability
    Type V: comminuted
    Type VI: fracture-dislocation
    • Schatzker classification
  • Mayo: based on displacement, stability and comminution
    Type I: nondisplaced (< 2 mm)
    IA: no comminution
    IB: comminution
    Type II: stable displaced (> 2 mm)
    IIA: no comminution
    IIB: comminution
    Type III: unstable, displaced fracture-dislocations (> 2 mm)
    IIIA: no comminution
    IIIB: comminution

  • OTA: radius/ulna proximal segment (21)
    Type A: extraarticular
    A1: ulna
    A2: radius
    A3: both
    Type B: articular, either radius or ulna
    B1: ulna fractured
    B2: radius fractured
    B3: articular in one, extraarticular in the other
    Type C: articular, both radius and ulna
    C1: simple fracture of both
    C2: simple fracture of one, comminuted fracture of the other
    C3: comminuted fractures in both
