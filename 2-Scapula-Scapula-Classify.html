Classification
  • Anatomic location:
    Body
    Glenoid neck
    Glenoid fossa
    Coracoid
    Acromion

  • Glenoid neck:
    Lateral to coracoid: while rare, these are inherently unstable
    Medial to coracoid: the glenoid/coracoid fragment is anchored to the axial skeleton via the CC ligament and the clavicle
    Type I: displaced < 1 cm, angulated < 40°
    Type II: displaced > 1 cm, angulated > 40°

  • Goss/Ideberg: glenoid fossa fractures
    Type IA: anterior glenoid rim
    Type IB: posterior glenoid rim
    Type II: transverse or oblique fracture exiting inferiorly producing inferior glenoid fragment
    Type III: transverse or oblique fracture exiting along superior scapular border producing superior glenoid fragment
    Type IV: horizontal exiting through medial border of scapular body and superior glenoid
    Type Va: type IV with type II
    Type Vb: multifragmentary type IV
    Type Vc: type Vb with type II
    Type VI: severe glenoid comminution
    • Goss/Ideberg classification
  • Acromion:
    Type I: minimally displaced
    Type II: displaced but acromial space maintained
    Type III: displaced with narrowing of the acromial space

  • Coracoid:
    Type I: proximal to CC ligaments
    Type II: distal to CC ligaments

  • OTA: scapula (14)
    Type A: extraarticular
    A1: acromion
    A2: corocoid
    A3: body
    Type B: partial articular (glenoid)
    B1: anterior rim
    B2: posterior rim
    B3: inferior rim
    Type C: complete articular (glenoid)
    C1: extraarticular glenoid neck
    C2: intraarticular with neck
    C3: intraarticular with body

  • OTA: scapulothoracic dissociation (10-D)
