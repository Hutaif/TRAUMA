Classification
  • Descriptive:
    Open or closed
    Skin condition: normal, tented, threatened
    Location: medial, middle, lateral
    Displacement: distance and direction
    Comminution
    Associated injuries including neurovascular compromise

  • Craig: modified Allman classification
    Group I: middle third
    Group II: distal third
    Type I: minimal displacement (interligament fracture)
    Type II: displaced, medial to CC ligaments
    IIA: conoid and trapezoid attached to distal segment
    IIB: conoid disrupted, trapezoid attached to distal segment
    Type III: articular surface fracture at AC joint
    Type IV: ligaments intact with periosteum and displacement of proximal fragment
    Type V: comminuted
    Group III: proximal third
    Type I: minimal displacement
    Type II: significant displacement
    Type III: intraarticular
    Type IV: ephiphyseal separation
    Type V: comminuted

  • Robinson-Edinburgh: based on displacement and comminution - both predictive of non-union
    Type 1A1: nondisplaced medial, extraarticular
    Type 1A2: nondisplaced medial, intraarticular
    Type 1B1: displaced medial, extraarticular
    Type 1B2: displaced medial, intraarticular
    Type 2A1: nondisplaced middle
    Type 2A2: angulated middle
    Type 2B1: displaced middle, simple or wedge
    Type 2B2: displaced middle, comminuted
    Type 3A1: nondisplaced lateral, extraarticular
    Type 3A2: nondisplaced lateral, intraarticular
    Type 3B1: displaced lateral, extraarticular
    Type 3B2: displaced lateral, intraarticular
    • Robinson-Edinburgh classification
  • OTA: clavicle (15)
    Type A: medial
    A1: extraarticular
    A2: intraarticular
    A3: comminuted
    Type B: metaphyseal
    B1: simple
    B2: wedge
    B3: complex
    Type C: lateral
    C1: extraarticular
    C2: intraarticular
